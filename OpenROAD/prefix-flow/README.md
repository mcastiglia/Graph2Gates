# Synthesis + PnR Flow Example

This example overviews performing synthesis and PnR on an ArithTreeRL-generated 32b adder.

## Files in this directory:
- `adder_32b.v`: Adder generated by ArithTreeRL
- `adder_nangate45_adder_32b.sdc`: Constraints file generated by ArithTreeRL
- `fast_flow.tcl`: ArithTreeRL fast PnR flow. Reports metrics and exits prior to routing
- `flow_helpers.tcl`: Taken from OpenROAD/test/ directory. Containers helper functions for PnR
- `full_flow.tcl`: ArithTreeRL PnR flow
- `helpers.tcl`: More helper functions taken from OpenROAD/test/ directory
- `NangateOpenCellLibrary_typical.lib`: Nangate45 technology library, taken from ArithTreeRL repo
- `openroad_flow.tcl`: OpenROAD flow script generated by ArithTreeRL
- `README.md`: This file
- `yosys_flow.tcl`: Synthesis flow script generated by ArithTreeRL

## Usage:

(Must be run within the Apptainer environment)

`apptainer shell --bind "$PWD":/workspace ./openroad.sif`

`cd prefix-flow`

`yosys -s yosys_flow.tcl`

`openroad openroad_flow.tcl`

Results and reports are added to the newly-created `results` directory.
